**1탄. Div 그리기**
===================

친구와 함께 'D3.js - data visualization' 책을 구매했다. 막연하게 데이터 시각화를 해보고 싶었는데, HTML과 css 기능을 사용한다는 점에서 Tableau와 같이 자체 앱으로 나온 시각화 툴과는 다르다는 것을 점점 더 느끼고 있다.

# 01_drawing divs

이 코드를 실행하면 청록색 막대기 한 개가 나온다.

```
<!DOCTYPE html>
<html lang='en'>
	<head>
		<meta charset = "utf-8">
		<title>D3 : Drawing divs </title>
        <script type="text/javascript" src="http://d3js.org/d3.v3.min.js"></script>
		<style type="text/css">

				div.bar {
					display: inline-block;
					width : 20px;
					height: 100px;
					background-color:teal;
			}

		</style>
	</head>
	<body>
		<script type="text/javascript">

			var dataset = [5, 10, 15, 20, 25];

			d3.select("body").selectAll("div")
			.data(dataset)
			.enter()
			.append("div")
			.attr("class","bar");

		</script>
	</body>
</html>
```


# 02_drawing divs_height 조절

이 코드를 실행하면 높이가 조절된 청록색 막대기 한 개가 나온다.

```
<!DOCTYPE html>
<html lang='en'>
	<head>
		<meta charset = "utf-8">
		<title>D3 : Setting div heights from data</title>
		<script type="text/javascript" src="http://d3js.org/d3.v3.min.js"></script>
		<style type="text/css">

				div.bar {
					display: inline-block;
					width: 20px;
					height: 100px;
					background-color: teal;
			}

		</style>
	</head>
	<body>
		<script type="text/javascript">

			var dataset = [5, 10, 15, 20, 25];

			d3.select("body").selectAll("div")
			.data(dataset)
			.enter()
			.append("div")
			.attr("class","bar")
			.style("height", function(d){
			return d + "px";
			});
		
		</script>
	</body>
</html>
			
```

# 03_drawing divs_space 조절

막대 여러 개의 간격이 일정하게 조정이 되어 나온다.
```
<!DOCTYPE html>
<html lang='en'>
	<head>
		<meta charset = "utf-8">
		<title>D3 : Setting margin </title>
        <script type="text/javascript" src="http://d3js.org/d3.v3.min.js"></script>
		<style type="text/css">

				div.bar {
					display: inline-block;
					width : 20px;
					height: 100px;
					margin-right : 2px;
					background-color:teal;
			}

		</style>
	</head>
	<body>
		<script type="text/javascript">

			var dataset = [5, 10, 15, 20, 25];

			d3.select("body").selectAll("div")
			.data(dataset)
			.enter()
			.append("div")
			.attr("class","bar")
			.style("height", function(d){
				var barHeight = d * 5;
				return barHeight + "px";
			});

		</script>
	</body>
</html>
```
